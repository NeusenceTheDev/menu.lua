# Feature List

## Core UI
- Centered in-game menu with animated open/close behavior
- Tabbed layout for Visual, Combat, World, and Settings
- Custom styling, gradients, rounded corners, and hover polish
- Hotkey support for menu toggling

## Diagnostics
- Error logging and command logging
- Global error handling for unhandled script errors
- Detailed script output and reporting helpers

## Weapon and Combat Tools
- Arsenal weapon data loading and caching
- Fire rate adjustments
- No recoil and spread reduction logic
- Kill-all controller and spawn tracking
- Fly toggle with input handling

## Visual Tools
- FOV and crosshair-related controls
- Chams and ESP-related menu support
- Lighting and brightness controls
- Custom sky and visual enhancement options

## World Tools
- Anti-flash and long melee options
- Miscellaneous movement and weapon behavior tweaks
- Environment-related menu controls

## Quality and Cleanup
- Idempotent feature toggles
- Connection cleanup on disable
- Startup orchestration with background initialization
